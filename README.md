Pulse — AI‑Powered Customer Feedback Insights for Small Businesses
Pulse is a lightweight tool that helps small business owners turn raw customer reviews into clear, actionable insights. By uploading a simple CSV file, users receive an instant summary of themes, complaints, compliments, and recommendations generated by an LLM.

Why this matters for small businesses
Small businesses collect feedback constantly—through receipts, Google reviews, Square receipts, social media, and in‑store conversations—but most owners don’t have the time or tools to analyze it. They end up with long lists of unstructured comments and no clear sense of what customers care about most.

Pulse focuses on three core needs:

Clarity: Owners want to know what’s working and what’s not without reading hundreds of reviews.

Speed: Insights need to be generated quickly so decisions can be made in real time.

Accessibility: Tools should be simple, affordable, and require no technical expertise.

By transforming raw text into structured insights, Pulse gives small businesses a fast, approachable way to understand customer sentiment and make better decisions—aligning directly with Square’s mission of empowering merchants with simple, powerful tools.

How constraints shaped the solution
Building Pulse required working within several practical constraints, which influenced the product decisions:

Free API usage:  
I used Groq’s free Llama‑3.1 model to avoid cost barriers while still providing fast inference. This shaped the prompt design and encouraged efficient text handling.

Simple, reliable deployment:  
Streamlit Cloud was chosen because it offers free hosting, easy GitHub integration, and secure secret management. This allowed me to focus on product behavior rather than infrastructure.

Dataset realism:  
I created a 100‑review dataset that mimics real restaurant feedback—balanced sentiment, varied topics, and natural language—so the demo produces meaningful insights without requiring proprietary data.

Scope discipline:  
To keep the project focused and shippable, I prioritized a single core workflow: upload → analyze → insights. Additional features (dashboards, charts, segmentation) were intentionally left for future iterations.

These constraints helped shape Pulse into a simple, focused tool that demonstrates product thinking, technical execution, and an understanding of small‑business needs.

How it works
User uploads a CSV containing a review column.

The app extracts and cleans the text.

The reviews are sent to Groq’s LLM with a structured prompt.

The model returns themes, complaints, compliments, recommendations, and a weekly summary.

Results are displayed instantly in the UI.
